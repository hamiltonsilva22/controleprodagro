# Sistema de Controle de Produção Agrícola

Este projeto é um sistema simples de controle de produção agrícola, desenvolvido em **HTML, CSS e JavaScript puro**, com foco em gerenciamento visual e simulação de controle de produção no campo.

## Funcionalidades
- Cadastro visual de talhões / áreas de plantio
- Acompanhamento de produção
- Interface simples e intuitiva
- Estrutura pronta para futura integração com banco de dados

##  Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
